## Description

## 👨‍💻 About Project
The Mini CRM Client Lead Management System is a web-based application designed to help businesses and freelancers manage client leads efficiently.

This project allows users to:
- Add new client leads
- Track lead status
- Update client information
- Delete unwanted leads
- Search and filter leads easily

The application is built using HTML, CSS, and JavaScript with LocalStorage support for storing lead data directly in the browser.

This project was developed as part of the Future Interns Full Stack Web Development Internship Task 2 (FUTURE_FS_02).

The system provides a clean and responsive user interface that works on both desktop and mobile devices.
