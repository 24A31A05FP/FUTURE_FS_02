Client Lead Management System (Mini CRM)

A fully offline, browser-native CRM built with pure HTML5, CSS3, and Vanilla JavaScript. All data is stored in browser LocalStorage — no database or server required. Deployable on Netlify or GitHub Pages instantly.

System Architecture

  • LocalStorage Engine: Full CRUD persistence in the browser — works offline.
  • Glassmorphism UI: Dark theme with backdrop blurs, gradient glows & animations.
  • Live Dashboard: Animated counters and pipeline funnel updated on every action.
  • Zero Dependencies: No Node.js, no MongoDB, no build step. Open index.html directly.
